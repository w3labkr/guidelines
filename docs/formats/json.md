# JSON

JSON (JavaScript Object Notation) is an open standard file format and data interchange format.

- [en.wikipedia.org/wiki/JSON](https://en.wikipedia.org/wiki/JSON){:target="_blank"}

## Example

```json
{
  "firstName": "John",
  "lastName": "Smith",
  "isAlive": true,
  "age": 27,
  "address": {
    "streetAddress": "21 2nd Street",
    "city": "New York",
    "state": "NY",
    "postalCode": "10021-3100"
  },
  "phoneNumbers": [
    {
      "type": "home",
      "number": "212 555-1234"
    },
    {
      "type": "office",
      "number": "646 555-4567"
    }
  ],
  "children": [],
  "spouse": null
}
```
