# stylelint-order

A plugin pack of order related linting rules for Stylelint.

- [github.com/hudochenkov/stylelint-order](https://github.com/hudochenkov/stylelint-order){:target="_blank"}

## Installation

```shell
npm install --save-dev stylelint-order
```

## Configuration

```json
{
    "plugins": [
        "stylelint-order"
    ],
    "rules": {
        "order/order": [],
    },
}
```
