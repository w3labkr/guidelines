## Software Versioning

Software versioning is the process of assigning either unique version names or unique version numbers to unique states of computer software.

Semantic versioning

```text
<Major>.<Minor>.<Patch>.<Status>
```

Calendar Versioning

```text
<Year>.<Month>.<Day>.<Status>
```
